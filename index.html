Quick Start

1. Test Locally

cd ~/Desktop/dms-social-poster
npm install
npm test   # Dry run - see what would post

2. Set Up Meta API

  1. Go to developers.facebook.com and create a new app (Business type)
  2. Add Facebook Login and Instagram Graph API products
  3. Use Graph API Explorer to get Page Access Token
  4. Note your Page ID and Instagram Business Account ID

3. Add GitHub Secrets

Secret NameValue
META_PAGE_ACCESS_TOKENYour Page Access Token
META_PAGE_IDYour Facebook Page ID
META_IG_USER_IDYour Instagram Business Account ID

Token Management

Important: Page Access Tokens expire after ~60 days. Set a calendar reminder to refresh.

For permanent tokens, use System User tokens via Business Manager.
